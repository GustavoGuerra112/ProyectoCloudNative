Analizador de Texto Distribuido

Como funciona el usuario escribe un texto, el sistema lo analiza y muestra cuantas palabras, caracteres y oraciones tiene. Todo corre en los contenedores DOCKER.

Las tecnologias que use son:
HTML,CSS y JavaScript para el frontend
FastAPI con Python para el backend
Redis para la cola de mensajes
Celery para los workers
Docker y Docker Compose
